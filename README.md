# HackerNews_clone

HackerNews_clone is a Hacker News clone (see https://news.ycombinator.com/) made with Ruby on Rails.

**Ruby version**<br />
2.1.5p273<br />
**Rails version**<br />
4.2.2<br />
**System dependecies**<br />
```$ bundle install```<br />
**Database creation**<br />
```$ rake db:migrate```<br />
**Database initialization[optional]**<br />
```$ rake db:seed```<br />
**Run Rails Application**<br />
```$ rails server```
